Cómo usar OrganizaT

Crear tarea: escribe un título, fecha, prioridad, etiquetas o descripción y presiona "Agregar". La tarea aparecerá en la columna de pendientes.

Arrastrar tareas: puedes mover una tarea entre columnas arrastrándola con el mouse, igual que en Trello. Así cambias su estado a pendiente, en progreso o completada.

Barra de progreso: arriba verás una barra que muestra el porcentaje de tareas completadas.

Buscador: el campo de búsqueda permite encontrar tareas por nombre, descripción o etiquetas.

Ordenar tareas: puedes ordenar por fecha, prioridad o nombre usando el menú de ordenamiento.

Modo oscuro: el botón de modo oscuro cambia el estilo visual para trabajar mejor en ambientes con poca luz.

Notificaciones: si una tarea vence hoy, el sistema mostrará una alerta automática.

Contador de tareas: cada columna muestra cuántas tareas tiene actualmente.
